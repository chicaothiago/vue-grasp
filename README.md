# Quasar App

## Pre-requirements

* [Nodejs](https://nodejs.org/en/)
* [npm](https://www.npmjs.com/)

> A Quasar project

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
